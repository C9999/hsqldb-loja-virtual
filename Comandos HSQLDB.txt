**Subir servidor do banco pelo CMD: 
java -cp hsqldb.jar org.hsqldb.server.Server --database.0 file:loja-virtual --dbname.0 loja-virtual

**Subir interface gráfica pelo CMD: 
java -cp hsqldb.jar  org.hsqldb.util.DatabaseManager

**Criar base de dados:
CREATE TABLE Produto (id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY, nome VARCHAR(255), descricao VARCHAR(255) )

**Insert de dados: 
insert into Produto values(1, 'Geladeira', 'Geladeira 110 com frezer e descongelamento automatico')

**URL de conexão: 
("jdbc:hsqldb:hsql://localhost/loja-virtual", "SA", "");
